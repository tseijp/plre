# API

## Set pl config

```ts
import { pl, createPL } from 'plre'
import { usePL } from 'plre/react'

pl({ x: [1, 2, 3], y: [1, 2, 3] }) // or

usePL({ x: [1, 2, 3], y: [1, 2, 3] }) // or

createPL({ x: [1, 2, 3], y: [1, 2, 3] })
```
